# C++ Menu Driven Mini Projects

This is a **menu-driven C++ console application** that combines multiple small programs into one project. It is designed for **practice and learning basic programming concepts** like conditions, loops, functions, and switch statements.

---

## 🚀 Features

This program includes the following functionalities:

1. Leap Year Checker
2. Day of the Week Finder
3. Height Category Calculator
4. Electricity Bill Calculator
5. Discount Calculator
6. Exam Eligibility Checker
7. Strong Number Checker
8. Traffic Light Simulator
9. Body Fat Calculator

---

## 🛠️ Concepts Used

* Functions
* Switch Case
* Conditional Statements (if-else)
* Loops (while)
* Basic Math Calculations
* User Input/Output

---

## 💻 How to Run

1. Install a C++ compiler (like g++, Dev C++, or CodeBlocks)
2. Clone this repository:

```bash
git clone https://github.com/talhaahmad56/cpp-practice-menu-app.git
```

3. Navigate to the project folder:

```bash
cd cpp-practice-menu-app
```

4. Compile the program:

```bash
g++ main.cpp -o app
```

5. Run the program:

```bash
./app
```
